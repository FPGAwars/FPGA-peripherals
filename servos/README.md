# Servos

Verilog PWM implementation. Using [Icezum](https://github.com/FPGAwars/icezum) board.

## Execute

Install [apio](https://github.com/FPGAwars/apio#installation).


| Action       | Command       |
|--------------|---------------|
| Verification | `apio verify` |
| Simulation   | `apio sim`    |
| Build        | `apio build`  |
| Upload       | `apio upload` |
| Clean        | `apio clean`  |
|              |               |
